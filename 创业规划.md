# 一人公司创业规划

> 创建时间：2026-03-17
> 核心方向：企业AI助理定制 + 产品化

---

## 一、创业理念

### 核心思路：像找工作一样创业

```
1. 基于能力找市场
   a. 先基于自己的爱好和特长，搜集相关的可做的岗位
   b. 然后在可做的岗位中，找到工资最高的几个。进行面试，尝试

2. 基于工作，更新能力
   a. 根据用户需求，不断精进自己的能力，提供更好的服务
   b. 能力扩展后扩大服务范围
```

### 关键洞察

- 岗位 ≠ 创业机会：岗位是雇佣逻辑，创业是交易逻辑
- 能力陷阱：只做自己擅长的，可能错过更大的机会
- 需要从"替代岗位"升级到"创造价值"

## 三、创业方向选择

### 方向对比

| 方向 | 收入潜力 | 产品化难度 | 复购性 | 被动收入 | 匹配度 |
|------|---------|-----------|--------|---------|--------|
| **企业AI助理定制** | ⭐⭐⭐⭐⭐ | 中 | 高 | 中 | ⭐⭐⭐⭐⭐ |
| **toB SaaS产品** | ⭐⭐⭐⭐⭐ | 高 | 高 | 高 | ⭐⭐⭐⭐ |
| **军事资讯分析服务** | ⭐⭐⭐ | 中 | 中 | 低 | ⭐⭐⭐ |
| **AI开发培训/咨询** | ⭐⭐⭐⭐ | 低 | 低 | 低 | ⭐⭐⭐⭐ |

### 最终选择：企业AI助理定制 + 产品化

**选择理由：**

1. 收入高：企业定制AI助理，单项目5-20万，维护费1-3万/月
2. 能力匹配：会配置OpenClaw + 懂toB产品设计 + 能快速开发
3. 产品化路径清晰：定制项目 → 提炼通用模块 → 行业模板 → SaaS化
4. 复购机制：月度维护 + 功能迭代 + 培训服务
5. 被动收入潜力：做成模板后，新客户成本极低

## 四、获客策略

### 获客途径全景

```
线下本地（青岛）
├── 人脉圈（前同事、朋友、行业聚会）
├── 企业拜访（直接上门）
├── 行业协会/商会
└── 创业园区/孵化器

线上平台
├── 猪八戒/一品威客（低价验证）
├── 闲鱼（个人客户）
├── 领英/脉脉（toB客户）
└── 抖音/小红书/B站（内容获客）

私域流量
├── 朋友圈
├── 微信群/社群
└── 公众号
```

### 青岛本地机会

**青岛产业结构：**
- 制造业（海尔、海信、中车等）
- 港口物流（青岛港、航运企业）
- 外贸企业（大量中小外贸公司）
- 旅游业（酒店、景区、旅行社）
- 教育培训（各类培训机构）
- 专业服务（律所、会计师事务所、咨询公司）

**推荐切入行业：外贸企业**

理由：
- 青岛外贸企业多（中小型为主）
- 痛点明确（客户沟通、订单管理、多语言）
- 有预算（外贸利润相对可观）
- AI应用场景清晰（客服、翻译、报价）

### 最佳获客路径

**首选：人脉圈 + 垂直行业切入**

```
第一步：梳理人脉
├── 列出所有做外贸的朋友/前同事
├── 列出所有在企业做管理的朋友
└── 列出所有在青岛创业的朋友

第二步：准备话术
"我最近在做企业AI助理定制，能帮企业做：
 - 自动回复客户询盘（多语言）
 - 订单状态查询
 - 产品知识库问答
 你身边有做外贸的朋友吗？可以帮我引荐一下"

第三步：提供免费诊断
"我可以免费帮你的朋友看看，他们的业务哪些环节可以用AI提效"
```

**备选：线上平台快速验证（猪八戒网）**

```
操作步骤：
1. 注册账号，完善资料
2. 搜索关键词：AI、智能客服、知识库、企业助手
3. 报价策略：前期低价（3000-8000），换取案例
4. 交付后要好评，积累口碑
```

### 执行计划

```
第1周：
├── 梳理人脉，发朋友圈/私信
├── 注册猪八戒，开始接单
└── 准备2-3个demo案例

第2-3周：
├── 跟进人脉介绍的机会
├── 完成猪八戒第一单
└── 整理案例，发朋友圈

第4周：
├── 用案例去拜访本地企业
├── 继续猪八戒接单
└── 开始做内容（公众号/视频号）
```

## 六、市场测试策略

### 核心思路：先卖概念，再开发

降低风险，快速验证市场需求。

### 测试方法

```
第1步：选择5-10个概念重点推广
├── 外贸询盘助手（青岛外贸多）
├── 培训招生助手（培训行业需求明确）
├── 电商客服助手（市场大）
├── 企业知识库助手（通用性强）
└── 律所咨询助手（客单价高）

第2步：多渠道投放
├── 朋友圈：每天发1个概念
├── 猪八戒：搜索相关需求，主动报价
├── 行业群：发概念卡片，看反馈
└── 视频号：做成短视频

第3步：收集反馈
├── 哪个概念问的人最多？
├── 哪个概念愿意付费？
├── 客户还提了什么需求？
└── 定价是否合理？

第4步：快速迭代
├── 热门概念 → 优先开发demo
├── 冷门概念 → 调整或放弃
└── 新需求 → 补充到概念库
```
